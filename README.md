# Проект: Прогнозирование оттока клиентов телеком-компании

## Описание
Проект посвящён анализу данных клиентов телекоммуникационной компании и построению модели машинного обучения для прогнозирования оттока клиентов. Решение позволяет заранее выявлять клиентов с высоким риском ухода и разрабатывать меры по их удержанию.

## Цель исследования
Построить модель прогнозирования оттока клиентов и определить ключевые факторы, влияющие на вероятность ухода, чтобы повысить лояльность клиентов и снизить финансовые потери компании.

## Ход исследования
- Предобработка и объединение данных из нескольких источников, включая отбор и создание новых признаков.
- Исследовательский анализ данных (EDA): анализ распределений, баланса классов и корреляций.
- Моделирование:
   - Подготовка данных к обучению моделей.
   - Обучение и сравнение моделей градиентного бустинга: LightGBM и CatBoost.
   - Оценка качества моделей и выбор лучшего решения.
   - Анализ матрицы ошибок, ROC-кривой и важности признаков.
- Итоговые выводы и рекомендации заказчику.

## Технологии
Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn, LightGBM, CatBoost, ROC-AUC, Feature Importance.

## Ключевые результаты
- Построены и сравнены модели LightGBM и CatBoost.
- Лучший результат показала модель LightGBM: значение ROC-AUC = 0.91 на тестовой выборке.
- Разработана модель, позволяющая выявлять клиентов с высоким риском оттока и использовать результаты для повышения эффективности программ удержания клиентов.
